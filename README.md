# Sistema de Cadastro de Usuários

Um sistema CRUD de cadastro de usuários. Cada um contendo id, nome, e-mail e telefone.



## Funcionalidades

 - CRUD de usuários

## Requisitos

 - PHP 8+
 - MySQL 8+

## Uso:
### Views:
 1. **Tela principal**
Assim que acessado, o sistema mostra uma lista com os usuários cadastrados.
